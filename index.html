Shine n Shimmer

This was a project created as an assignment while still at SPC.... We were given a choice of ten topics to choose from. We were then tasked with creating a site for the business type we chose. The 'owners' was placed as my sister and brother-in-law. This was, I think, the third site I built.

SPC Issue Log

A final of sorts for a JavaScript class I had while still attending SPC.... The premise given by the instructor was a bit confusing. It boiled down to being able to create a new user, save that information to localStorage, then get redirected to a log in page. You'd log in with that info, and be taken to another page where you could really enter any input, save that as well. Saved input would then be displayed and able to be deleted if unwanted.

Summit Massage

Fur Baby Boutique

This was prior to responsive design being explained. Again, a false business created solely for class and entry experience.

Project Job 87
